Twitchの広告をブロックする方法　TTV LOL PRO編

1. TTV LOL PROの使い方ガイド

TTV LOL PROは、Twitchのライブストリームから広告をブロックするための便利なツールです。このガイドでは、TTV LOL PROのインストール方法と使用方法について詳しく説明します。

2. TTV LOL PROの概要

TTV LOL PROは、Twitchの広告を削除するためのブラウザ拡張機能です。無料で利用でき、完全ではないものの、多くの広告を効果的にブロックします。

3. インストール方法

Chromeの場合

  1. ダウンロード: GitHubの リポジトリ からTTV LOL PROのソースコードをダウンロードします。
  2. 解凍: ダウンロードしたファイルを解凍します。
  3. 拡張機能の設定:
    • Chromeを開き、アドレスバーに chrome://extensions と入力します。
    • 右上の「デベロッパーモード」を有効にします。
    • 「未パッケージ化の拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

Firefoxの場合

  1. ダウンロード: GitHubのリポジトリからTTV LOL PROのソースコードをダウンロードします。
  2. 拡張機能の設定:
    • Firefoxを開き、アドレスバーに about:debugging#/runtime/this-firefox と入力します。
    • 「一時的なアドオンを読み込む」をクリックし、ダウンロードしたファイルを選択します。

Microsoft Edgeの場合

  1. ダウンロード: GitHubのリポジトリからTTV LOL PROのソースコードをダウンロードします。
  2. 解凍: ダウンロードしたファイルを解凍します。
  3. 拡張機能の設定:
    • Edgeを開き、アドレスバーに edge://extensions と入力します。
    • 左下の「デベロッパーモード」をオンにします。
    • 「解凍された拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

4. 使用方法

TTV LOL PROをインストールした後、特別な設定は必要ありません。Twitchのストリームを視聴する際に、自動的に広告がブロックされます。

uBlock Originとの併用

TTV LOL PROをさらに効果的に使用するために、uBlock Originと併用することもできます。以下の手順で設定を行います。

  1. uBlock Originのインストール: ChromeまたはFirefoxの拡張機能ストアからuBlock Originをインストールします。
  2. フィルターの追加:
    • uBlock Originのダッシュボードを開きます。
    • 「マイフィルター」タブに移動し、以下のフィルターを追加します。
    twitch.tv##+js(twitch-videoad)
    • 「設定」タブで「私は上級ユーザーです」を有効にし、表示される歯車アイコンをクリックします。
    • userResourcesLocation の値を、使用したいスクリプトのURLに変更します。例えば：
    userResourcesLocation https://raw.githubusercontent.com/pixeltris/TwitchAdSolutions/master/video-swap-new/video-swap.js

5. 注意事項

  • TTV LOL PROは無料で利用できますが、完璧な広告ブロックを保証するものではありません。
  • Twitchの利用規約に違反しないように注意してください。

まとめ

TTV LOL PROを使用することで、Twitchの広告を効果的にブロックし、快適な視聴体験を得ることができます。インストールと設定は簡単で、特にuBlock Originとの併用が推奨されます。ぜひ試してみてください。
